# 锦城新闻安卓APP
这是一个校园新闻APP

## 技术环境
* IDE：Android Studio 3.1.3
* Gradle：gradle-4.4-all.zip
* Android API: 20~27 (Android 4.4w ~ Android 8.1)

## 关键点
* Fragment
* TabLayout + ViewPager
* [GreenDAO](https://github.com/greenrobot/greenDAO)
* [OkHttp](https://github.com/square/okhttp)
